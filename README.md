# Quant Alchemy Skills

Open-source agent skills for AI assistants.

## Skills

### do-parallel-work
Parallel task queue — orchestrate work across multiple sub-agents.

### greptile-worktree
Git worktree workflow for Greptile code review on unindexed branches. Use when implementing features that need Greptile review but the target branch isn't indexed.

## Installation

These skills are designed for use with [ClawdHub](https://clawdhub.com) and compatible AI agent frameworks.

```bash
clawdhub install quantalchemy/do-parallel-work
clawdhub install quantalchemy/greptile-worktree
```
